Not Supported!

Although I understand this is a very basic and desirable feature, it is not supported yet!

Read on below to understand about the techincal difficulties about supporting your login password updation feature. Since all your password's sensitive info is encrypted, it is done so using your login password as a master key. Changing this means, we need to go through all your passwords, decrypt them and encrypt with your new login password. This is too much of migration work for a simple change, and would raise a reliability issue. To migitate login password changes to some extent, I will be adding MFA based two-factor login soon as a measure of additional security.
